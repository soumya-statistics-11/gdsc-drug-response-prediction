# AI-Powered Precision Oncology: Cancer Drug Response Prediction and Recommendation System

## Overview

This project develops an AI-driven system to predict cancer drug response and recommend potential therapeutic options using the Genomics of Drug Sensitivity in Cancer (GDSC) dataset.

The objective is to classify whether a cancer cell line is **Sensitive** or **Resistant** to a particular anti-cancer drug and rank drugs based on their predicted effectiveness. The project combines Machine Learning, Deep Learning, Explainable AI, and a Drug Recommendation Engine to support precision oncology research.

---

## Dataset

**Dataset:** Genomics of Drug Sensitivity in Cancer (GDSC)
**Link** https://drive.google.com/file/d/1UfMzAR2myigIRtdXhl0xyY2qwJPmdKsF/view?usp=drive_link

* 242,035 observations
* 19 variables
* Cancer cell line information
* Drug information
* Tissue descriptors
* Drug targets and pathways
* Pharmacological response measurements

### Target Variable

A binary target variable (**Response**) was created using LN_IC50 values:

* Sensitive (1): LN_IC50 below median
* Resistant (0): LN_IC50 above median

## Model Performance

| Model               | Accuracy   | ROC-AUC    |
| ------------------- | ---------- | ---------- |
| Logistic Regression | 82.29%     | 0.9105     |
| Random Forest       | 81.50%     | 0.8983     |
| XGBoost             | 81.13%     | 0.9036     |
| CatBoost            | **83.45%** | **0.9216** |
| Neural Network      | 83.24%     | 0.9183     |

**Best Model:** CatBoost

* Explainable AI provided insights into the biological factors driving model predictions.
* The recommendation engine successfully ranked drugs according to predicted sensitivity.
